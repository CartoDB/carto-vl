Class: Interactivity

carto.Interactivity(layerList)

new Interactivity(layerList)

Interactivity purpose is to allow the reception and management of user-generated events, like clicking, over layer features.

To create a Interactivity object an array of carto.Layer is required. Events fired from interactivity objects will refer to the features of these layers and only these layers.

Parameters:
Name Type Description
layerList carto.Layer | Array.<carto.Layer>

carto.Layer or array of carto.Layer, events will be fired based on the features of these layers. The array cannot be empty, and all the layers must be attached to the same map.
Fires:
  • event:featureClick
  • event:featureClickOut
  • event:featureHover
  • event:featureEnter
  • event:featureLeave
  • event:CartoError
Example
const interactivity = new carto.Interactivity(layer);
interactivity.on('click', event => {
  style(event.features);
  show(event.coordinates);
});
